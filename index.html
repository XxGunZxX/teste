Abrir Supabase ↗
🔑 Credenciais do Supabase
Informe os dados do seu projeto. Você encontra essas informações em Settings → API no painel do Supabase.
Como obter as credenciais:
  1. Acesse app.supabase.com e abra seu projeto
  2. No menu lateral, vá em Settings → API
  3. Copie a Project URL e a chave anon public
Formato: https://[id-do-projeto].supabase.co
Chave pública JWT — segura para uso no front-end
🗄️ Configurar Banco de Dados
Execute o schema SQL abaixo no seu projeto Supabase. Isso criará as tabelas, índices, políticas RLS e triggers necessários.
Execução automática via Anthropic + Supabase MCP.
O wizard vai conectar ao Supabase e executar o schema automaticamente. Para isso, informe sua Service Role Key (Settings → API → service_role secret).
⚠️ Nunca exponha esta chave no front-end do app. Usada somente aqui para setup.
Passos para execução manual:
  1. Clique em Copiar SQL abaixo
  2. Abra o SQL Editor no painel do Supabase
  3. Cole e clique em Run
  4. Volte aqui e clique em Concluído
Abrir Supabase SQL Editor ↗
